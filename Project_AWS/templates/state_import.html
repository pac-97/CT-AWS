Import Behavior

      
Import is read-only. It creates a state document by querying selected AWS service modules.

      

        

          
StepWhat happens

          
            
1. Select modulesChoose services like IAM, EC2, S3, Organizations, Identity Center, CloudTrail.

            
2. Query AWS APIsCollector runs each selected module independently.

            
3. Build unified snapshotReturns one JSON with `meta` + `services` sections.

            
4. Handle partial accessIf one service permission fails, error is recorded under `meta.errors` and others continue.

            
5. Use in workflowsCopy JSON to compare drift and build future plan/apply modules.
